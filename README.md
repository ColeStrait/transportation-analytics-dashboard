# Transportation Analytics Dashboard

A Spring Boot web application for analyzing transportation materials testing data. The application imports CSV files, stores the data in an H2 database, and displays interactive charts and summary statistics.

## Features

- CSV file import
- Material Test Data table
- Density and Air Voids charts
- Analysis Summary
- Key Statistics Dashboard
